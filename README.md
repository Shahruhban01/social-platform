# Real-Time Social Audio Platform

A production-scale backend platform powering a real-time social audio application featuring live audio rooms, user interactions, virtual gifting, authentication, and content management.

> **Note:** This repository serves as a portfolio showcase of the system architecture, technologies, and engineering contributions. The production source code is proprietary and belongs to the employer, therefore it is not publicly available.

---

# Overview

This project focused on designing, developing, and maintaining scalable backend systems supporting real-time communication and high-concurrency user interactions.

The platform provides:

* Live audio broadcast rooms
* Real-time messaging and reactions
* Virtual gifting system
* Authentication & user management
* In-app virtual economy
* Admin management portal
* Media management
* Notification system
* VIP membership system
